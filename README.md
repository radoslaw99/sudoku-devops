# Sudoku DevOps

Prosty projekt Sudoku napisany w HTML, CSS i JavaScript.
Repozytorium zostało utworzone w ramach laboratorium z przedmiotu DevOps.

## Cel projektu
Celem projektu jest zaprezentowanie podstawowego workflow DevOps:
- praca z Git i GitHub
- użycie kluczy SSH
- praca na branchach
- konteneryzacja aplikacji (Docker)
- automatyzacja (GitHub Actions – w kolejnym etapie)

## Technologie
- HTML
- CSS
- JavaScript
- Git / GitHub

## Uruchomienie
Aby uruchomić aplikację:
1. Sklonuj repozytorium
2. Otwórz plik `index.html` w przeglądarce
